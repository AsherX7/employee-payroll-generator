# employee-payroll-generator

## Project Description

Employee Payroll Generator is a Java-based desktop application developed using Java Swing and MySQL. The system helps manage employee records, calculate salaries, generate payslips, and process payroll efficiently.

## Features

- Employee Record Management
- Add Employee
- Update Employee
- Delete Employee
- Search Employee
- Payroll Generation
- Payslip Generation
- Salary Calculation
- Full-Time Employee Management
- Part-Time Employee Management
- Tax and Allowance Management
- MySQL Database Integration
- JDBC Connectivity

## Technologies Used

- Java
- Java Swing
- JDBC
- MySQL
- Eclipse IDE

## Database

Database: payroll_schema

Tables:
- employee
- payroll
- employee_allowances

## How to Run

1. Clone the repository.
2. Import the project into Eclipse.
3. Configure MySQL database.
4. Update database credentials in DBConnection.java.
5. Run LoginFrame.java.
